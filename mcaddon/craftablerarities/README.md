# CraftableRarities

**稀有物品可合成** — 让 Minecraft 中各种稀有、不可再生、非法/技术性方块都能通过工作台合成！

> **版本**: 0.0.2 | **适用**: Minecraft Bedrock 1.21+

---

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 构建 + 打包 .mcaddon
npm run mcaddon

# 输出位置：dist/packages/craftablerarities-v*.mcaddon
```

## 🛠️ 常用命令

```bash
npm run build          # 版本同步构建
npm run mcaddon        # 构建 + 打包为 .mcaddon
npm run clean          # 清理构建产物
npm run local-deploy   # 部署到本地 Minecraft 开发目录

# 版本管理
npm run version:patch  # 0.0.1 → 0.0.2
npm run version:minor  # 0.0.1 → 0.1.0
npm run version:major  # 0.0.1 → 1.0.0
```

## 👀 创造物品栏

本 Add-on 同时集成了**隐藏方块创造物品栏补充**功能，让你在创造模式中可以直接获取以下特殊方块，无需使用命令：

- 屏障、隐形的基岩、结构方块、结构空位
- 命令方块（普通/连锁/循环）
- 传送门、末地传送门、折跃门
- 下界反应核、发光黑曜石
- 静态水、静态岩浆
- 气泡柱、火、灵魂火
- 移动方块、活塞头
- 以及更多技术性方块和完整半砖/按钮/告示牌变种
