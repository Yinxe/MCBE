# @yinxe/mc — MCBE Addon

> Minecraft Bedrock Edition Addon — Monorepo

## 项目结构

```
.
├── apps/           # 应用程序/Addon 项目
├── packages/       # 共享库和工具包
├── pnpm-workspace.yaml
└── package.json
```

## 要求

- **Node.js** >= 20
- **pnpm** >= 11.1.3

## 快速开始

```bash
# 安装依赖
pnpm install

# 构建所有包
pnpm -r run build
```
